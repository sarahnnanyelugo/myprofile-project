My Projects

Yeah, I work hard. Each project is unique. Here are some of my works:

⭐Airline Booking System

This program simulates booking seats on a flight. It includes features like displaying available seats, reserving a seat, canceling a booking, and storing passenger details. I also implemented file handling to save and retrieve bookings for persistence.

⭐Contacts Manager

A small system for storing, retrieving, updating, and deleting contact information (like name, phone, and email). It uses file storage (e.g., CSV ) with dynamic memory management (malloc, realloc, free). It’s served as a good introduction to CRUD operations, file handling, and dynamic data structures in C.

⭐Coin Game

This is a simple 1D game where a player tries to “catch” coins as they appear randomly. Typically involves generating random values for coin positions, letting the player decide moves, and tracking scores.This project helped strengthen my understanding of concepts like random number generation, loops, and conditional logic.

⭐Students Grader

This is a program that takes student names and scores, then calculates grades based on given criteria (A, B, C, etc.). It also computes averages, highest/lowest scores, and display results in a structured way. This project reinforces my understanding of arrays, loops, and conditional branching.
